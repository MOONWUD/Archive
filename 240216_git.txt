오후에는 git, github를 배움 
아무런 정보가 없을때 
git config부터 설정 (이름이랑 이메일을 github랑 동일하게 설정해야함)
git config --list로 잘 저장되었는지 확인

로컬과 온라인 이름을 다르게 설정해도 됨 

온라인에서 만든 것을 로컬로 내려 받는 방법 
 ​git clone 내 주소 ~
내가 지금 있는 곳이 저장소가 맞는지 확인
들어가야할 파일로 이동해서 (main) 이 옆에 붙는지 확인 ! 
git pull origin main 

이때부터 로컬에서 수정 가능 
git pull (branch가 1개일 경우 이렇게만 쳐도 가능)
